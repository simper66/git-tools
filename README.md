# git-tools

Set of git hooks, daemons, guards, etc, such as:
 
- encrypt/decrypt before/after push/pull.
- automatic up/down sync of files/folders.
- warn/prevent contributors from editing some content.
- chained scripts as rebase/squash/push
- parametrized scripts as reset "3" commits behind and "not" preserve workspace. 
- automatized integrations and configurations with other applications/systems.
- act as a not proprietary cloud storage for floss applications like notes or bookmark apps.
- use as a central configuration storage.
